# Match-3

A browser-based match-3 puzzle game built with vanilla HTML/CSS/JS — no dependencies, single file.

## Features

- 8×8 grid, 6 gem types
- Swap animation with invalid-move feedback
- Cascade combos with score multiplier
- Falling animation with physics-based duration
- Sound effects via Web Audio API
- Auto-play bot

## How to Play

Open `match3.html` in any modern browser. Click two adjacent gems to swap them. Match 3 or more in a row/column to score. Combos multiply your points.

## Run

```bash
open match3.html
```
